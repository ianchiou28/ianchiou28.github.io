# 📸 照片相册

## 使用方法

### 1. 放入照片
把照片文件放入这个 `/pictures` 文件夹

### 2. 配置文件名
打开根目录的 `main.js`，找到最开头的 `photoFiles` 数组，填入你的照片文件名：

```javascript
const photoFiles = [
    'photo1.jpg',
    'photo2.png',
    // 添加更多...
];
```

### 3. 刷新网页
保存文件，刷新网页即可看到照片

---

## ⚠️ 注意事项

- 支持的格式：`.jpg` `.jpeg` `.png` `.gif` `.webp`
- 照片会公开显示，请勿上传隐私照片
- 建议单张不超过 2MB
